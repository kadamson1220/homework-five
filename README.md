# Homework Five - Travel-Fly Website

This project is a multi-page responsive website for "Travel-Fly," a fictional travel agency. It showcases various destinations, special offers, client testimonials, and a contact page.

## Features

- **Responsive Design**: The website is fully responsive, ensuring a seamless experience across devices of all sizes.
- **SCSS for Styling**: SCSS is used for modular and maintainable styles, with separate files for each page and components.
- **Font Awesome Integration**: Includes Font Awesome icons for social media links and other design elements.
- **Multi-Page Layout**: Contains multiple pages, including:
  - Home
  - About
  - Tours
  - Special Offers
  - Blog
  - Contact
